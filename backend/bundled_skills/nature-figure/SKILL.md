# Nature Figure

Use this skill for scientific figures that must support a manuscript-style conclusion.

## Hard Constraints

- Always define the figure's conclusion, evidence chain, panel purpose, and target export formats before generating publication-style figures.
- Always keep every panel tied to a specific claim, comparison, or quality-control decision.
- Always check labels, units, legends, sample sizes, color meaning, statistical annotation, and accessibility before calling a figure final.
- Must export vector formats such as SVG or PDF for plots when feasible, and high-resolution raster only when the content requires it.
- Must report visual review status for every generated figure artifact.
- Do not use decorative chart elements that obscure the scientific comparison.
- Do not present a figure as publication-ready when statistical methods, groups, labels, or visual review are missing.

## Figure Review

Assess whether the viewer can recover the main conclusion without reading the full chat. Fix clutter, overplotting, unreadable labels, missing legends, and unsupported significance marks.
