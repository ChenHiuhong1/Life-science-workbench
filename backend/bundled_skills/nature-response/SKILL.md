# Nature Response Skill

This is a repository-bundled response skill inspired by common high-impact journal revision practice. It does not depend on private local skills.

## Hard Constraints

- Always separate editor comments, reviewer comments, author responses, manuscript changes, and remaining limitations.
- Always answer every reviewer point explicitly.
- Always mark where the manuscript was changed, using section, figure, table, page, or line placeholders when exact locations are unknown.
- Must distinguish performed analyses from proposed future work.
- Must not invent new experiments, data, statistical results, editor positions, or reviewer intent.
- Do not become defensive; keep responses precise, evidence-grounded, and constructive.

## Response Structure

1. Opening letter: concise thanks, summary of major revisions, and tone of cooperation.
2. Global change summary: main experiments, analyses, text revisions, figures, data availability updates.
3. Point-by-point response:
   - Reviewer comment.
   - Response.
   - Manuscript change.
   - Evidence or limitation.
4. Final closing statement.

## Revision Quality Checks

- Each response must state whether the manuscript was changed.
- Each claim must map to evidence, analysis, citation, or a clearly labeled limitation.
- Statistical and methodological changes must include enough detail for reproducibility.
